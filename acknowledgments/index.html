鸣谢

感谢以下优秀的开源项目、服务和个人，让这个博客得以存在和持续改进。

🛠️ 技术栈

静态网站生成器

  • Hugo - 世界上最快的网站构建框架，让内容创作变得简单高效
  • Go - Hugo 的底层语言，简洁而强大

前端技术

  • Tailwind CSS - 实用优先的 CSS 框架，让样式开发更加灵活
  • Alpine.js - 轻量级的 JavaScript 框架，提供响应式交互
  • Heroicons - 精美的 SVG 图标库

部署与托管

  • Netlify - 现代化的 Web 开发平台，提供持续部署和全球 CDN
  • GitHub - 代码托管和版本控制，开源协作的基石
  • Cloudflare - 全球网络服务，提供安全和性能优化

🎨 设计与资源

字体

  • Inter - 专为屏幕显示优化的现代字体
  • JetBrains Mono - 专业的等宽编程字体
  • Noto Sans CJK - Google 的开源中日韩字体

图片与图标

  • Unsplash - 高质量的免费图片资源
  • Picsum Photos - 随机占位图片服务
  • Emoji - 丰富的表情符号，让内容更生动

🧰 开发工具

编辑器与 IDE

  • Visual Studio Code - 强大的代码编辑器
  • Windsurf - AI 驱动的开发环境

版本控制

  • Git - 分布式版本控制系统
  • GitHub Desktop - 简化 Git 操作的桌面应用

包管理与构建

  • Node.js - JavaScript 运行时环境
  • npm - Node.js 包管理器

📚 学习资源

文档与教程

  • MDN Web Docs - 权威的 Web 开发文档
  • Hugo Documentation - 详尽的 Hugo 使用指南
  • Tailwind CSS Documentation - 完整的 Tailwind 使用文档

社区与论坛

  • Hugo Community - Hugo 官方社区论坛
  • Stack Overflow - 程序员问答社区
  • GitHub Discussions - 项目讨论和交流

🌟 特别感谢

开源贡献者

感谢所有为开源项目贡献代码、文档和想法的开发者们。正是你们的无私奉献，让整个技术生态系统变得更加美好。

内容创作者

感谢那些分享知识、经验和见解的博主、作者和教育者们。你们的内容启发了我，也让我有动力去分享自己的学习心得。

读者朋友们

感谢每一位访问这个博客的读者。你们的阅读、反馈和支持是我持续创作的最大动力。
