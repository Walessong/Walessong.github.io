# 仓耳今楷02W4 字体

本站**中英文**统一使用 **仓耳今楷02W4**（即仓耳今楷02-W04，TsangerJinKai02-6763-W04）。

## 下载字体

1. **免费字体网**（推荐，获取下载次数后免费下载）  
   https://www.mianfeiziti.com/thread-75798.htm  
   - 页面中点击「下载字体」获取 **仓耳今楷02-W04** 的 TTF 文件（约 3.92 MB）。  
   - 下载前请关闭浏览器「阅读模式」。

2. 将得到的 TTF 文件重命名为：  
   **`CangErJinKai02W4.ttf`**  
   并放入本目录：`static/fonts/`。

3. **可选**：为更好兼容与加载速度，可再生成网页用字体：
   - **WOFF2**：推荐，体积更小。可用 [CloudConvert](https://cloudconvert.com/ttf-to-woff2) 将 TTF 转为 woff2，保存为 `CangErJinKai02W4.woff2` 放入本目录。
   - **WOFF**：同上方式转为 woff，保存为 `CangErJinKai02W4.woff` 放入本目录。

只要存在 **`CangErJinKai02W4.ttf`**，站点即可正常显示；若同时提供 woff2/woff，会优先使用以加快加载。
