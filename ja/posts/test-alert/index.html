高度な使用のヒント

この折りたたまれたヒントボックスには高度な使用のヒントが含まれています：

これらのステップを順番に実行することを忘れないでください。

折りたたみ可能な警告アラート

セキュリティ警告

続行する前に、以下のセキュリティ考慮事項に注意してください：

使用例

ドキュメント説明

注意

この機能はバージョン2.0から利用可能です。古いバージョンを使用している場合は、まずアップグレードしてください。

ベストプラクティス

パフォーマンス最適化の推奨事項

最高のパフォーマンスを得るために推奨されます：

  • 静的リソースにCDNを使用する
  • gzip圧縮を有効にする
  • 画像サイズを最適化する
  • ブラウザキャッシュを使用する

重要な設定

重要

設定ファイルを変更した後、変更を有効にするためにサービスを再起動する必要があります。

セキュリティリマインダー

警告

本番環境でデフォルトパスワードを使用しないでください。

破壊的操作

注意

この操作はすべてのデータを永続的に削除し、復元できません。重要なデータをバックアップしていることを確認してください。

これらのアラートは、ユーザーがコンテンツの重要性とコンテキストをよりよく理解するのに役立ち、ドキュメントの可読性とユーザーエクスペリエンスを向上させます。

15class ApiClient {
16  private baseURL: string;
17  private headers: Record<string, string>;
18
19  constructor(baseURL: string, apiKey?: string) {
20    this.baseURL = baseURL;
21    this.headers = {
22      'Content-Type': 'application/json',
23      ...(apiKey && { 'Authorization': `Bearer ${apiKey}` })
24    };
25  }
26
27  async get<T>(endpoint: string): Promise<ApiResponse<T>> {
28    const response = await fetch(`${this.baseURL}${endpoint}`, {
29      method: 'GET',
30      headers: this.headers,
31    });
32
33    if (!response.ok) {
34      throw new Error(`HTTPエラー！ステータス：${response.status}`);
35    }
36
37    return response.json();
38  }
39
40  async post<T>(endpoint: string, data: any): Promise<ApiResponse<T>> {
41    const response = await fetch(`${this.baseURL}${endpoint}`, {
42      method: 'POST',
43      headers: this.headers,
44      body: JSON.stringify(data),
45    });
46
47    return response.json();
48  }
49}
